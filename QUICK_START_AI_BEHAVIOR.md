# ⚡ Quick Start - AI Behavior Detection

## 🚀 Chạy thử tính năng trong 5 phút

### Bước 1: Setup (1 phút)
```bash
cd Interview_it_gottalent
flutter pub get
```

### Bước 2: Run App (2 phút)
```bash
flutter run
```

### Bước 3: Test tính năng (2 phút)
1. Mở app → Login/Register
2. Chọn "Luyện tập Phỏng vấn" hoặc "Thuyết trình"
3. Upload bất kỳ file PDF nào
4. Nhấn "Bắt đầu luyện tập"
5. **AI Badge xuất hiện ở góc trên bên trái!** 🤖

### 📱 Xem AI hoạt động

**Badge sẽ thay đổi dựa trên hành vi của bạn:**

#### Test 1: Tập trung ✅
- Nhìn thẳng vào camera
- Giữ đầu thẳng
- **Kết quả**: Badge hiển thị "🎯 Tập trung" (màu xanh)

#### Test 2: Cười 😊
- Mỉm cười tự nhiên
- **Kết quả**: Badge hiển thị "😊 Đang cười" (màu xanh)

#### Test 3: Mất tập trung 👀
- Quay đầu sang trái/phải
- **Kết quả**: Badge hiển thị "👀 Mất tập trung" (màu đỏ)

#### Test 4: Cúi đầu 📱
- Cúi đầu nhìn xuống
- **Kết quả**: Badge hiển thị "📱 Cúi đầu" (màu đỏ)

#### Test 5: Nghiêng đầu ⚠️
- Nghiêng đầu sang một bên
- **Kết quả**: Badge hiển thị "⚠️ Nghiêng đầu" (màu vàng)

### 📊 Xem thống kê

1. Nhấn icon **📊** ở góc dưới bên phải
2. Panel thống kê hiển thị:
   - **Điểm tập trung**: 0-100
   - **% Tích cực**: Hành vi tốt
   - **% Tiêu cực**: Hành vi cần cải thiện
   - **5 hành vi gần nhất**: Timeline
3. Nhấn **✕** để đóng

### 🎮 Toggle Controls

**Bật/Tắt AI:**
- Nhấn **"🤖 AI ON/OFF"** (góc trên trái)
- AI có thể tắt bất cứ lúc nào

**Bật/Tắt Panel:**
- Nhấn **📊** (góc dưới phải)
- Xem/ẩn thống kê nhanh chóng

## 🐛 Troubleshooting

### "Không thấy người" ❓
**Fix**: 
- Kiểm tra camera có bật không
- Di chuyển vào giữa khung hình
- Tăng ánh sáng

### Badge không xuất hiện
**Fix**:
- Nhấn "🤖 AI ON" để bật
- Restart app
- Check camera permission

### Phát hiện không chính xác
**Fix**:
- Cải thiện ánh sáng
- Điều chỉnh khoảng cách camera (30-100cm)
- Đảm bảo mặt không bị che
