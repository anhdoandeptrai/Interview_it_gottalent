  • Rx<bool> - Boolean states

┌─────────────────────────────────────────────────────────────────────────────┐
│                        🛡️ ERROR HANDLING                                    │
└─────────────────────────────────────────────────────────────────────────────┘

  Every operation has try-catch:
  
  try {
    [Operation]
  } catch (e) {
    [Log error]
    [Show user message]
    [Fallback or retry]
  }

  Fallback hierarchy:
  1. Gemini AI ✓
  2. OpenAI (if configured)
  3. Predefined questions (last resort)
